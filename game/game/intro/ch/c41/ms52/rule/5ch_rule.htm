五子棋規則與開局

一、簡易規則

  1.由黑棋先行，黑白雙方輪流落子在棋盤的交叉點。
  2.先在棋盤上形成橫、直、或縱的連續五個棋子者（連五）為勝。

由於簡易規則下，執黑者有必勝的著手，故開始有黑方禁著的規定（日本規則）。

二、日本規則（簡稱日規）

  1.行棋方式同簡易規則。
  2.黑方有三三、四四及長連禁手（包含四三三、三三四…），在連成五子前，
    無論主動或被動下出禁著點，立即判定為負。換言之，黑方唯一獲勝的可
    能為下出四三。
  3.白方無任何禁點，長連亦為勝。
  4.五連與禁手同時形成則不算禁手，黑方可算獲勝。

三、國際規則

  1.行棋方式同簡易規則。
  2.禁手規定同日式規則。
  3.猜子決定假先手。
  4.假先手持黑，雙方共下三子，輪到白方時，假後手此時有選擇先後手之權
    利，真正先後手在此底定，稱為「三手交換」。若假後手認為開局對黑方
    有利，可提出交換成為先手；反之亦可不交換，選擇繼續當後手。因此，
    假先手開局必須選擇黑白雙方局勢均衡之開局，否則這盤棋將大為不利。
  5.第四手由後手白方自由落一子，先手黑方此時必須提供兩個黑子著點，供
    後手白方任選其一，稱為「五手兩打」。此後黑白雙方開始自由落子。

也就是說，假先手選擇開局，假後手選擇真正的先後手（因為開局後有三子，
故稱為三手交換）。之後，後手再下第四子，先手提出第五手的兩種選擇給後
手（故稱五手兩打），然後雙方便自由行棋。

四、開局

  

第一子黑子下在天元，第二子白子下在天元為中心的3x3區域內，第三子黑子下
在天元為中心的5x5區域內，扣除同型後，前三手共有26種不同的情況，所以稱
為26開局。

目前已知黑方最易獲勝的開局法為I7浦月、D4花月或D6雨月（同I6雲月）。

第二子白子若不貼著黑走，下在天元為中心的3x3區域內，基本上大都必敗。至
於第三子黑子若不下在以天元為中心的5x5範圍內，只要下在白子那一側，不要
離太遠，大多數是黑優。
