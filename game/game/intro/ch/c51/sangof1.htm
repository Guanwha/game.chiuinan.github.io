名稱：三國志武將爭霸 (Sango Fighter，中文版)
簡介：熊貓1993年出品的格鬥遊戲，在早期相當有名。遊戲中有統一天下、武將爭
      霸、雙人對決等三種模式。統一天下採格鬥攻城略地的方式進行，玩家扮演
      劉備，帶領五虎將攻佔各地以統一天下。而在這個模式中敵方會先由小嘍嘍
      出陣，然後才由大頭頭出陣，己方則由五虎將中挑選一人，陣亡即換人。武
      將爭霸則以打敗遊戲中的十二位角色為目標，採積分排行榜制。雙人對決則
      是由玩家兩人相互對決。整個遊戲設計得還不錯，喜愛格鬥遊戲的人可以玩
      玩看。
 
保護：數字密碼，破解碼：
      SANGO.EXE (需用EXESHAPE解壓)
        E8 71 68 2E 3B 1E 1B 07 75 E5
        2E 8B 1E 1B 07 89 1E DE 67 90
      或
        A0 00 07 32 E4 03 D8
        90 2E 8B 1E 1B 07 90
密技：1.只要將檔案的*.SPR互相改檔名，即可使用其他人。例如GG.SPR把他
        拷貝成AA.SPR，進遊戲選關羽即會變成夏候淵。至於檔案列表自己找
        吧，如：
          AA.SPR => 關羽
          BB.SPR => 張飛
          CC.SPR => 趙雲
          DD.SPR => 馬超
      2.快速破關法－首先進入GB或FPE，接著再進入遊戲，然後進入統一天
        下單元，馬上用熱鍵叫出GB或FPE，用低階功能找出體力的值將之鎖
        定，回到遊戲依序按下Esc、A、Tab進行臨陣脫逃，則馬上進到下一
        關！不出三分鐘就可看到全破畫面了！（鎖住位址：C630h）
修改：氣不減法
      SANGO.EXE (需用EXESHAPE解壓)
        38 16 00 00  (左邊控制，計２次)
        -- -- -- 01
        3A 16 00 00  (右邊控制，計２次)
        -- -- -- 01
