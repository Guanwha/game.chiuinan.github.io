名稱：大富翁2 (中文版)
簡介：大宇1993年出品的大富翁遊戲。遊戲中除了購買基本的土地，以及類似
      命運機會的新聞外，還增加了如賭博、接錢、買賣股票等賺錢的方式，
      並取消了繞一圈拿錢的方法，因此遊戲過程中的行進方向並不固定。更
      特別的是增加了特殊用途的卡片，以及神明的附身，增加不少遊戲的趣
      味與難度。遊戲的過程主要是經過台灣、香港、台北三地，打敗電腦所
      控制的敵人，成為首富。只是這款遊戲仍不免有電腦自行做手腳之嫌，
      電腦控制的人物大都能以稍高的機率前進到空地上購買土地，還好遊戲
      提供了存取檔功能，玩家也能利用存取大法來平衡此一弱點。整體而言，
      這款遊戲的娛樂性相當不錯，喜歡玩大富翁遊戲的玩家可以玩玩看。

*** 一般資料解壓法

no = 讀取2 byte整數，低位元組在前
處理下列解壓程序no次
{
  b = 讀取1 byte
  c = (b低7位元) + 1
  if b最高位元=1 then
    讀取c byte資料輸出
  else
    b = 讀取1 byte
    輸出b共c次
  end if
}

*** 圖像解壓法

xsize = 圖檔水平大小
ysize = 圖檔垂直大小
處理ysize個水平圖像
{
   s = xsize
   while s > 0
   {
     b = 讀取1 byte
     c = b低7位元
     if b最高位元=1 then 輸出c byte背景色
     else 讀取c byte輸出
     s -= c
   }
}
