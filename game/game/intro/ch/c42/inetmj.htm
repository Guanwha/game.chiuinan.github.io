﻿名稱：哈拉麻將王 (中文版)
簡介：向量科技2001年出品的16張麻將遊戲，主要目的是提供連線與其他人打
      麻將，與電腦對戰只是遊戲的額外功能，因此並未提供協助玩家的道具，
      也不會特別給玩家好牌與進張的機會，完全靠運氣與牌技決勝。遊戲共
      提供16位角色做為個人競技的打牌對象，電腦的牌技不差，要獲勝確實
      需要運氣。幸好遊戲可隨時存檔，玩家可藉由存取檔來調整打法以取得
      優勢，因此算是一款可拿來訓練牌技的遊戲。



保護：不明
備註：若遊戲無法順利啟動，可試著改放在目錄C:\INETMJ裡。
修改：存檔INetMjS?.CKC

位置0860h = 各家籌碼，均4 byte
位置0934h開始為手中的牌，每張4 byte，共16張。各牌的代號為：
  11(0Bh)～19(13h) = 1～9萬
  21(15h)～29(1Dh) = 1～9索
  31(1Fh)～39(27h) = 1～9筒
  41(29h)～44(2Ch) = 東南西北
  51(33h)～53(35h) = 發中白

操作：滑鼠右鍵 = 吃碰槓胡選單
說明：

1.個人競技賽，玩到籌碼不足的人即退出，換人上場並重新開局。一直玩到牌
  腳不足為止。
2.打牌若面臨抉擇時，可存檔後再打，若覺抉擇的結果不太好時，可讀檔回來
  換另一種打法試試看。
